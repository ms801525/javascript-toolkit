# 🛠️ Utility Toolkit

A collection of useful web-based tools built using **HTML**, **CSS**, and **JavaScript** to practice core front-end development concepts including DOM manipulation, event handling, form validation, responsive design, and JavaScript logic.

---

## 📖 Project Overview

The Utility Toolkit is a beginner-friendly JavaScript project consisting of four interactive utilities:

- 🧮 Simple Calculator
- 📏 Unit Converter
- 📝 Text Analyzer
- {} JSON Formatter

The purpose of this project was to strengthen JavaScript fundamentals while building practical, interactive tools.

---

## 🚀 Features

### 🧮 Simple Calculator

- Addition
- Subtraction
- Multiplication
- Division
- Exponent (Power)
- Square Root
- Input validation
- Prevent division by zero
- Press **Enter** to calculate
- Clear button

---

### 📏 Unit Converter

Supports multiple conversions including:

- Kilograms ↔ Pounds
- Pounds ↔ Kilograms
- Kilometers ↔ Miles
- Miles ↔ Kilometers
- Celsius ↔ Fahrenheit
- Fahrenheit ↔ Celsius
- Meters ↔ Feet
- Feet ↔ Meters
- Liters ↔ Gallons
- Gallons ↔ Liters
- Inches ↔ Centimeters
- Centimeters ↔ Inches

Additional Features:

- Input validation
- Enter key support
- Clear button

---

### 📝 Text Analyzer

Analyze text with useful statistics including:

- Word Count
- Character Count
- Sentence Count
- Most Common word

Additional Features:

- Multi-line text input
- Resizable text area
- Clear button

---

### {} JSON Formatter

Generate properly formatted JSON from user input.

Features include:

- Dynamic key-value pairs
- Pretty formatted JSON output
- Copy to clipboard
- Clear button
- Input validation
